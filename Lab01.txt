1. Change Directory:
cd dir (Change directory to dir)

2. Create a Directory:
mkdir dir (Make directory dir)

3. Rename a Directory: 
mv old_name new_name (change old_name to new_name)

4. List the contents of a directory: 
ls (list files)

5. Move a file to a new location: 
mv file1 dir (Move file1 to dir)

6. Copy a file to a new location: 
cp file1 dir (Copy file1 to dir)

7. Search for a string in a file: 
grep string file1 (search for string in file1)

8. Access the shell of a remote machine: 
ssh remote_host (secure shell [domain name])

9. Remove a directory folder: 
rm -rf dir (remove files and folders in dir)
